# vue-fade-carousel

## Demo

https://syumai.github.io/vue-fade-carousel/dist/

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Building bundle for production
```
yarn run build-bundle
```

### Building demo app
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```
